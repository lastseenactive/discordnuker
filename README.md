# 🤖 Discord Nuker Setup Guide

Follow these steps to get your bot up and running quickly.

---

## 🔑 1. Add Your Bot Token

1. Open the file:

   ```
   token.txt
   ```
2. Paste your Discord bot token inside:

   ```
   YOUR_BOT_TOKEN_HERE
   ```
3. Save the file.

---

## ⚙️ 2. Enable Required Intents

1. Go to the **Discord Developer Portal**
2. Open your application
3. Navigate to the **Bot** tab (left sidebar)
4. Scroll to **Privileged Gateway Intents**
5. Enable the following:

* ✅ Presence Intent
* ✅ Server Members Intent
* ✅ Message Content Intent

---

## ▶️ 3. Run the Bot

After completing setup, simply **double-click**:

```
setup.bat
```

---

## 🛠️ 4. Troubleshooting

**Bot not working? Check the following:**

* ✔️ Your token is correct and properly pasted
* ✔️ Required intents are enabled
* ✔️ The bot has been invited with the correct permissions

---
